# Blocked In Combat 惊天矿工团
一款 Minecraft 经典小游戏玩法。本项目是一个基于此玩法复刻的 Spigot 插件，适用于 Minecraft Java Edition 1.21+ 版本。
## 游戏基本玩法
> 游戏分为 4 个队伍。 游戏开始后会先生成一个 32x32 的地图， 这 32x32 的空间中由一些方块组成。 四个队伍会被分配到地图的四个角落。每个队伍的目标通过获取地图上的资源来合成装备、武器等，并最终击败其他队伍的玩家，最后剩下的队伍获胜。
